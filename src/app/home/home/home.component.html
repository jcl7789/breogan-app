Este es mi proyecto final del Profesorado Superior en Informatica.

¿Que es?

Es una SPA (Single page application), diseñada y desarrollada con fines academicos. Para la misma, utilice el Stack MEAN, compuesto por 4 tecnologias: MongoDB, Express, Angular y NodeJS. Si bien, esta pensada como dos aplicaciones distintas, una en front-end y una en back-end, es posible deployarla como si fuera una sola.

Inicialmente, fue concebida como una aplicacion para un server local, de uso solo para los empleados. Actualmente, se puede escalar a un servidor web, permitiendo la consultar o modificar la informacion, segun el role asignado por el PO. (Product Owner)

¿Que podria ser?

Una solucion integral para comercios, con modulos de manejo de inventario, turnos, estadisticas...

¿Cual es la utilidad de esta página?

Una vez el sitio se encuentre productivo, será la página de presentación del comercio.
